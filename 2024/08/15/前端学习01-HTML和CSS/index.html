1. 基础知识概览

CSS 盒模型（Box Model）是网页布局的核心概念之一。每一个 HTML 元素都被看作是一个矩形的盒子，盒模型决定了这个盒子的大小以及它在页面中占据的位置。盒模型由以下几个部分组成：

  1. 内容区域（Content Area）：这是盒子中实际内容所在的区域，内容的大小通常由元素的 width 和 height 属性决定。

  2. 内边距（Padding）：这是内容区域和边框之间的空白区域，内边距是透明的，不会改变背景颜色。

  3. 边框（Border）：这是包裹内容和内边距的边框，可以设置不同的样式、颜色和宽度。

  4. 外边距（Margin）：这是盒子与其他元素之间的空白区域，外边距也是透明的，可以用来控制元素之间的距离。
